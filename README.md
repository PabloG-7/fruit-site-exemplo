# 🌿 Avora Mart

> Plataforma de e-commerce para groceries frescos com design premium e experiência mobile-first.

🔗 **Demo:** https://pablog-7.github.io/fruit-site-exemplo/

---

## 🎯 Sobre o Projeto

**Avora Mart** é uma aplicação web completa desenvolvida com HTML, CSS e JavaScript puro. O projeto foi criado para demonstrar um e-commerce moderno com navegação intuitiva, design responsivo e uma experiência de usuário fluida.

## ⚡ Funcionalidades

### 🔐 Login
- Design fullscreen com imagem hero
- Transição suave para o dashboard

### 🏪 Shop
- Banner promocional com ofertas
- Categorias de produtos
- Flash Deals com timer
- Grid de produtos com favoritos e carrinho
- Página de detalhes do produto

### 📦 Orders
- Lista de pedidos com status
- Filtros por status
- Barra de pesquisa
- Modal de detalhes
- Ações rápidas (Reordenar, Rastrear)

### 💬 Chat
- Lista de conversas
- Indicador de online
- Badge de mensagens não lidas

### 👤 Profile
- Avatar com edição
- Informações do usuário
- Menu de configurações
- Logout

## 🎨 Destaques do Design

- **Glassmorphism** - Efeito blur em headers e navegação
- **Animações Suaves** - Pulse, float, fade, bounce
- **Mobile First** - Design adaptado para todos os dispositivos
- **Badges Inteligentes** - Notificações claras em telas menores
- **Scrollbar Customizada** - Estilização premium
